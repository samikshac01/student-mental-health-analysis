# 🧠 Student Mental Health Analysis

## 📊 Objective
Analyze factors affecting student mental health such as academic pressure, sleep, and financial concerns.

## 🛠️ Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib

## 🔍 Key Insights
- Academic pressure increases anxiety
- Low sleep leads to higher depression
- Financial stress impacts mental health
- High-risk students identified (low sleep + high pressure)

## 📌 Conclusion
Student mental health is influenced by multiple lifestyle and academic factors. Balanced routines can help improve well-being.
